Will's Dice Game Rules

You are playing against the computer in a dice game. Five dice will be rolled in a competition to see who can get the highest point total. After the first roll, you will be able to select dice to keep, and re-roll the remaining dice to try for a higher total. A total of 3 rolls are allowed.

Player name rules:

  • Username must be letters only, and between 4 and 8 characters inclusive
  • Each character in the username must be alphabetically bigger than the letter before (e.g. abcd is a valid username, but adcb is not)
  • The username must start with an uppercase letter. If you don't enter the first letter as a capital, I will fix this for you.
